#### You can pass input default attributes

You can add the default attributes like `required` `disabled` `checked` on the element and they will be added to the input.

<Canvas>
  <Story name="attributes">
    {{
      components: { ReCheckbox },
      template: `
      <div>
        <ReCheckbox id="checked-1" checked="checked" label="Checked"/>
        <ReCheckbox id="indeterminate-1" checked="checked" :indeterminate="true" label="Indeterminate"/>
        <ReCheckbox id="disabled-1" disabled="disabled" label="Disabled"/>
        <ReCheckbox id="disabled-checked-1" checked="checked" disabled="disabled" label="Disabled checked"/>
      </div>
    `,
    }}
  </Story>
</Canvas>
